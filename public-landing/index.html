How It Works

Get started in three simple steps.

1

Start from the Landing Page

Click Start Using Zamanak on this homepage. That opens the auth entry on hamgam.zamanak24.ir/start.html, which calls the same php/hamgam/button.php used in the Paziresh24 panel and then opens Google OAuth if needed.

2

Connect Google Calendar

Click "Connect Google Calendar" and authorize Zamanak through Google's OAuth consent screen.

3

Configure Sync Preferences

Choose event color, select which appointment fields appear in calendar events, and save your settings. New appointments sync automatically.

Google Calendar Access

Information for Google OAuth verification and users.

Zamanak requests access to Google Calendar to create and manage calendar events related to user appointments. Calendar data is used only for appointment synchronization.

The application uses the following Google Calendar permission:

https://www.googleapis.com/auth/calendar.events

This permission allows Zamanak to:

  • Create appointment events in Google Calendar when bookings are made.
  • Update existing appointment events when booking details change.
  • Remove events when appointments are canceled.
  • Maintain synchronization between appointments and Google Calendar events.

Zamanak does not sell user data, use data for advertising, or share Google Calendar information with third parties.

Users can revoke access at any time through their Google Account permissions page or by disconnecting inside the app.
